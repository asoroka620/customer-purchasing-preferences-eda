# Customer Purchasing Preferences EDA

This project investigates the factors that influence customer purchasing preferences using exploratory data analysis (EDA). Through statistical testing, visualizations, and feature exploration, the analysis examines how demographic and seasonal factors impact both product category preferences and specific item purchases.

## Research Question
What factors influence customer purchasing preferences?

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SciPy

## Key Findings
- Gender significantly influences product category preferences.
- Age does not significantly influence broad product category preferences.
- Age does not influence preferences for specific products.
- Season influences specific product choices, although broad product category preferences remain consistent across seasons.
